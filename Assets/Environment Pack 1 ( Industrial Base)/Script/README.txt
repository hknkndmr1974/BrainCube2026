Thank you for buying my pack. I'll tell you about the main features of the GridSnapper script and how to use it.

Main features:
	- Automatic binding of objects to a customized grid
	- Snap when moving objects in the editor
	- Customizable grid cell size and offset
	- Step-by-step rotation of objects at a given angle
	- Visual prompts in the editor (Gizmos)
	- Automatic script deletion during the game for optimization

Component Parameters:

	Grid Settings
	- Grid Size - grid cell size (default: 1)
	- Offset - grid offset relative to the world (default: 0,0,0)

	Binding Settings
	- Snap While Moving - whether to snap an object to the grid when moving (default: enabled)
	- Snap On Start - snap an object to the grid when the scene starts (default: off)
	- Destroy On Play - delete the script when starting the game (saves resources, by default: enabled)

	Rotation Settings
	- Rotation Step - rotation step in degrees (usually 45 or 90)
	- Rotate Clockwise - direction of rotation (true - clockwise, false - counterclockwise)

	Managing in the editor

	Buttons in the inspector
	- ↺ Rotate by X° - rotate the object by a preset angle with reference to the grid
